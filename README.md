# DoveAPI

快速、简洁、非MVC

- 运行快(虚拟主机上最快0.0015秒，在一台2核服务器上运行速度最快0.000998秒)

![dove](http://dove.xcenadmin.top/run_time.png)

- 可扩展(extend)
- 方便二次开发(请留版权)
- 完美支持一个API所需的操作(request、response)
- 非MVC架构
- 访问控制
- 自带Postman
